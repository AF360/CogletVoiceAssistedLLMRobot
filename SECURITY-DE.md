# Sicherheitsrichtlinie

Wenn du ein Sicherheitsproblem in diesem Projekt entdeckst, öffne bitte **kein** öffentliches Issue.

Kontaktiere mich stattdessen bitte per E-Mail: **enquiries@acelab.net**.
Ich prüfe die Meldung und antworte so bald wie möglich.

Aktuell gibt es keine Authentifizierung und keine Upload-Limits für die STT-/LLM-Nutzung, um DoS oder Ähnliches zu verhindern, da der Code für den Betrieb in einem privaten Heimnetz gedacht ist.
